🏦 Cash Flow (CF)

File Excel theo template CF — header dòng 6, dữ liệu từ dòng 7. Cột O = Bộ phận (3 ký tự đầu = mã dự án), I = Phát sinh nợ → F30, J = Phát sinh có → F31. Cột F chứa "hoàn ứng/hoàn tạm ứng" → giá trị cột I đổi âm.

Import Hướng dẫn nhanh

  • File Project List (PL): cần có sheet tên "Project List", dữ liệu từ dòng 4 (row 4)
  • File Receivable: sheet "Receivable", dòng 5 = header, dữ liệu từ dòng 6 — Cột A = Project Name, D = Contract excl.VAT → F17, E = Contract incl.VAT → F22, F = Workdone excl.VAT → F25, L = Payment Request → F35, M = Received → F36 (SUMIF theo tên dự án)
  • File CF (template): header dòng 6, data từ dòng 7 — Cột O = Bộ phận (lấy 3 ký tự đầu làm mã dự án), I = Phát sinh nợ → F30, J = Phát sinh có → F31. Dòng cột F có keyword "hoàn ứng/hoàn tạm ứng/hoàn tiền tạm ứng" → cột I chuyển thành âm
  • File Revenue (template): header dòng 6, data từ dòng 9 — Cột M = Vụ việc (mã dự án, full code), J = Phát sinh có → SUMIF → F40
  • File Expenses (template): header dòng 6, data từ dòng 9 — Cột O = Bộ phận (lấy 3 ký tự đầu làm mã dự án), I = Phát sinh nợ, J = Phát sinh có → SUMIF (I − J) → F41
  • Các file CSV: dùng dấu phẩy (,) làm dấu ngăn cách

Hướng dẫn deploy email server lên Render.com (miễn phí):
1. Tải file email-server.zip từ nút bên dưới
2. Đăng ký render.com → New Web Service → Upload code
3. Vào Environment Variables điền: SMTP_HOST, SMTP_PORT, SMTP_USER, SMTP_PASS
4. Deploy xong → copy URL dán vào ô Server URL bên trên

Khi import file Excel, data tự động lưu lên cloud.
Tất cả người dùng pull về để lấy data mới nhất.
📖 Hướng dẫn lấy Supabase credentials
1. Vào supabase.com → đăng nhập → chọn project pmr-system
2. Menu trái → Settings → API
3. Copy Project URL (dạng https://xxxxxxxx.supabase.co)
4. Copy anon public key (chuỗi dài bắt đầu bằng eyJ...)
5. Dán vào 2 ô bên trên → nhấn Kết nối
6. Nhấn Tạo Tables (chỉ cần làm 1 lần)
7. Từ đó mỗi lần import file → data tự lên cloud ✓
